Mistra Node: Autonomous Security & Analytics
Mistra Node — це дипломний проєкт студента ДПУ (м. Ірпінь), що являє собою багатофункціональний автономний вузол у форматі Discord-бота.

Основні можливості:

AI-Intelligence: Інтеграція Mistral AI з використанням RAG-архітектури для обробки локальних баз знань.

OSINT Security: Автоматизований аналіз URL-адрес (VirusTotal API, Whois) з розрахунком авторського показника Risk Score.

Crypto Monitoring: Аналіз ринкових настроїв та котирувань у реальному часі через Binance API.

Stress Testing: Вбудований лабораторний модуль для аудиту продуктивності системи (CPU, RAM, Latency).
